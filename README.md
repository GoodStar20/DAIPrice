# DAIPrice


## Getting Started

### Clone this repo
```bash
$ git clone https://github.com/GoodStar20/DAIPrice.git
```

### Install dependencies
```bash
$ cd DAIPrice
$ yarn install
```

### Run the App on iOS
And then, run the app
```bash
$ yarn ios
```

### Run the App on Android

And then, run the app
```bash
$ yarn android
```
